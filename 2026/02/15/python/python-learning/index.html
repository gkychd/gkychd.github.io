# python 编程从入门到实践

  1. python 的数据类型主要分为：列表、元组、字典；

    1. 列表：中括号构造，[‘math’, ‘English’, Chinese’]，可随意修改内部的成员。
    2. 元组：小括号构造，dimensions = (50, 200)，无法通过访问成员去修改值，但可以通过重新构造的方式修改
    3. 字典：大括号构造，{‘color’ : ‘green’, ‘point’ : 5}，分为 key 和 value。

    sort：永久排序，cars.sort ();
